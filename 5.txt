Куб любого натурального числа n равен сумме n нечетных чисел, следующих по порядку за числами, сумма которых составила куб числа n−1. С помощью этого факта составьте таблицу кубов натуральных чисел от 1 до 100.


#Найти квадрат данного натурального числа, если квадрат любого натурального числа n равен сумме n первых нечетных чисел.

def square(n):
    count = 0
    for i in range(1, n ** 2):
        if i % 2 != 0:
            count += i
            while count ==  n ** 2:
                 return count
n = int(input())
print(square(n))
